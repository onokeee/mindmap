# Flask マインドマップアプリ

Flaskベースのインタラクティブなマインドマップ作成アプリケーションです。

## 機能

- ✨ ドラッグ&ドロップでノードを移動
- ➕ ノードの追加・削除
- ✏️ テキストの編集
- 💾 マインドマップの保存・読込
- 🎨 美しいグラデーションデザイン
- 🔗 ノード間の接続線を自動描画

## セットアップ

1. 必要なパッケージをインストール:
```bash
pip install -r requirements.txt
```

2. アプリケーションを起動:
```bash
python app.py
```

3. ブラウザで以下のURLにアクセス:
```
http://localhost:5000
```

## 使い方

### 基本操作

- **ノードの選択**: ノードをクリック
- **ノードの移動**: ノードをドラッグ
- **キャンバスの移動**: 空白部分をドラッグ
- **テキスト編集**: テキストをダブルクリック

### ノードの操作

- **子ノードを追加**: ノードにマウスを乗せて ➕ ボタンをクリック
- **ノードを削除**: ノードにマウスを乗せて ✕ ボタンをクリック

### データの管理

- **保存**: 上部メニューの「💾 保存」ボタンをクリック
- **読込**: 上部メニューの「📂 読込」ボタンをクリック
- **リセット**: 上部メニューの「🔄 リセット」ボタンをクリック

## ファイル構成

```
.
├── app.py                 # Flaskアプリケーション本体
├── requirements.txt       # 依存パッケージ
├── templates/
│   └── index.html        # フロントエンドHTML
└── data/
    └── mindmap.json      # 保存データ（自動生成）
```

## 技術スタック

- **バックエンド**: Flask (Python)
- **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla)
- **データ保存**: JSON形式

## カスタマイズ

### デザインの変更

`templates/index.html` の `<style>` タグ内でCSSを編集できます。

### ポート番号の変更

`app.py` の最終行を編集:
```python
app.run(debug=True, host='0.0.0.0', port=5000)  # ← ここを変更
```
